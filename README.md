# 💼 Budget Boss

A simple and intuitive budgeting web application that helps users track income, manage expenses, and stay in control of their finances.

## 📌 Features

* Add and track income 💰
* Add and categorize expenses 🧾
* Real-time balance updates 📊
* Transaction history table 📅
* Expense breakdown visualization (chart) 📉
* Responsive and user-friendly interface

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Chart.js (for data visualization)

---

## 🚀 How to Run Locally

1. Clone the repository:

```
git clone https://github.com/De-Bongz/Budget-Boss-app.git
```

2. Open the project folder:

```
cd Budget-Boss-app
```

3. Open `index.html` in your browser
